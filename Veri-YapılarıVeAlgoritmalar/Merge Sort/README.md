# Merge Sort Projesi

[16,21,11,8,12,22] -> Merge Sort

1. Yukarıdaki dizinin Merge Sort türüne göre aşamalarını yazınız.

    - [16,21,11]    [8,12,22]
    - [16] [21,11]  [8] [12,22]
    - [16] [21] [11] [8] [12] [22]
    - [16] [11,21] [8] [12,22]
    - [11,16,21] [8,12,22]
    - [8,11,12,16,21,22]


2. Big-O Notation
    - Worst-Case = o(nlogn)
    - Best-Case = o(nlogn)
    - Average case = o(nlogn)
